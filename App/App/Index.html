AngularJS Tetris


by Theo Kandiliotis

This is an original AngularJS + Plain Javascript version of the most popular video game ever.

It is built using the following technologies:

  • AngularJS 1.6.4
  • Bootstrap 3.3.7
  • Javascript
  • WebAPI
  • MySql + Entity Framework

Control

Use the arrow keys LEFT, RIGHT to move the tetromino, UP to rotate and DOWN to accelarate. If you are using a mobile device, a virtual on-screen keyboard will appear.

Source code

The full source code is available for download on my Github account. The project was created with Microsoft Visual Studio 2015 on September 2017.
